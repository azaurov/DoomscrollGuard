DoomscrollGuard

Vibrates your Amazfit GTR 3 when you've been doomscrolling.

Android only. DoomscrollGuard is a native Android app — it needs an AccessibilityService and "draw over other apps" permission that a web page cannot grant. This site is an installable shortcut/info hub, not the app itself. There is no iOS build; iOS does not expose the accessibility APIs this project relies on.

How it works

  1. An AccessibilityService on your phone counts scroll events in social apps.
  2. Scroll past your threshold within the time window → the app fires a notification.
  3. The Zepp app mirrors that notification to your GTR 3 → the watch vibrates.

No content is ever read — only scroll event counts.

Install the app

  1. Download the APK from the Releases page (or build it yourself from source in Android Studio) and sideload it.
  2. Open DoomscrollGuard, tap Enable Accessibility Permission, and turn it on for DoomscrollGuard.
  3. Grant the overlay ("draw over other apps") permission when prompted.
  4. In the Zepp app: Profile → GTR 3 → Notifications → enable App Alerts, and make sure DoomscrollGuard is toggled on.
